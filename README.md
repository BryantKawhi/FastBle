# BleHelper
Android Bluetooth Low Energy

- 支持与外围BLE设备进行扫描、连接、读、写、通知订阅与取消等基本操作
- 支持获取信号强度、设置最大传输单元
- 支持自定义扫描规则
- 支持多设备连接
- 支持重连机制
- 支持配置超时机制
